# 📚 Book Review System

A web-based application developed as part of the Web Technologies course.
The system allows users to browse books and manage customer reviews.

## 🎯 Project Description

This project is a dynamic website that demonstrates the use of:

* HTML & CSS for structure and design
* JavaScript for validation and interactivity
* PHP & MySQL for backend and database operations

The system allows users to:

* Add customer reviews
* View all reviews
* Update existing reviews
* Delete reviews

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

* MySQL

## ⚙️ Features

* 📖 Book browsing interface
* 🔍 Search functionality using JavaScript
* 📝 Customer review form
* ➕ Insert new reviews
* 📋 Display all reviews from database
* ✏️ Update reviews
* ❌ Delete reviews

---

## ▶️ How to Run Locally

1. Install XAMPP
2. Move the project folder to:
   C:\xampp\htdocs\
3. Start Apache and MySQL
4. Open in browser:
   http://localhost/project1/

---

## 🗄 Database Setup

* Create a database named: `bookreview`
* Import the SQL file (if included)
* Table used: `review1`
